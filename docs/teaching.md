# Teaching

## Courses

### Data Programming - Universum College

This course teaches core programming concepts in Python with emphasis on real data manipulation tasks from science, engineering, and business. By the end of the course, students can independently implement useful programs from a data source and problem description.

Terms: WiSe 2021/22

### [Lab "Distributed Big Data Analytics"](https://github.com/SmartDataAnalytics/MA-INF-4223-DBDA-Lab) (MA-INF 4223) - University of Bonn

The lab provides hands-on experience with big data processing tools such as [Apache Spark](http://spark.apache.org/) and introduces functional programming patterns used in concurrent and parallel big data systems.

Terms: SoSe 2019, WiSe 2018/19, SoSe 2018, WiSe 2017/18, SoSe 2017
